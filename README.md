# Interactive Navigation Menu

A clean, responsive landing page example with a fixed navigation bar that changes style on scroll and highlights menu links on hover.

## Files

- `index.html` — page structure and navigation menu
- `styles.css` — layout, responsive styling, and scroll/hover transitions
- `script.js` — scroll detection and hover interaction logic

## Features

- Sticky header navigation
- Smooth style transition when scrolling
- Menu link highlight on hover
- Mobile-friendly responsive menu layout

## Usage

1. Open `index.html` in a browser.
2. Scroll down the page to see the navigation bar change style.
3. Hover over menu links to view the highlight effect.

## Notes

- This demo is designed for learning and prototyping simple interactive navigation behavior.
- You can extend it with animated active link states, a mobile hamburger menu, or section scroll spy behavior.
